# Date dot Time

Package for showing date, time, etc.

# Options

- date

- time

- datetime

- day

- month

- random

# Time

Example:

```js
const datedottime = require('datedottime');

console.log(datedottime.time);

```

# Date

Example:

```js
const datedottime = require('datedottime');

console.log(datedottime.date);

```

# Date and Time

Example:

```js
const datedottime = require('datedottime');

console.log(datedottime.datetime);
```

# Day of the week

const datedottime = require('datedottime');
