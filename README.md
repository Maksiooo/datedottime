# Day dot Time

Package for showing date, time, etc.

# Options

- date

- time

- datetime

- day

- month

- random

# Time

Example:

```js
const daydottime = require('daydottime');

console.log(daydottime.time);

```

# Date

Example:

```js
const daydottime = require('daydottime');

console.log(daydottime.date);

```

# Date and Time
